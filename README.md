# 🎓 University Management System

A Java-based **University Management System** connected with a database.  
This project is developed for university semester submission using **Object-Oriented Programming (OOP)** concepts and database integration.

---

## 🚀 Features

- Add Student Records  
- Update Student Details  
- Delete Student Records  
- Search Students  
- Manage Courses  
- Manage Faculty (if included)  
- Database connectivity  
- Simple and user-friendly GUI (if Swing used)

---

## 🛠️ Technologies Used

- Java (Core Java / OOP)  
- Swing (GUI, if used)  
- MySQL Database  
- JDBC (Java Database Connectivity)  
- IntelliJ IDEA (IDE)

---

The project uses a MySQL database to store all records.
